#markdown 

# 마크다운 문법 

---

## 1. 제목 (Headings)

- `#` 개수로 제목 수준 표현
    

```markdown
# 제목1
## 제목2
### 제목3
#### 제목4
```

---

## 2. 강조 (Emphasis)

- 기울임: `*텍스트*` 또는 `_텍스트_`
    
- 굵게: `**텍스트**`
    
- 굵게 + 기울임: `***텍스트***`
    

```markdown
*기울임*
**굵게**
***굵고 기울임***
```

---

## 3. 목록 (Lists)

### 3.1 순서 없는 목록

- `-`, `*`, `+` 사용
    
- 들여쓰기로 하위 목록 구성
    

```markdown
- 항목1
- 항목2
  - 하위항목
```

### 3.2 순서 있는 목록

- 숫자 + `.` 사용
    

```markdown
1. 첫 번째
2. 두 번째
3. 세 번째
```

---

## 4. 링크 (Links)

- `[텍스트](URL)` 형식
    

```markdown
[OpenAI](https://openai.com)
```

---

## 5. 이미지 (Images)

- `![대체텍스트](이미지URL)` 형식
    

```markdown
![이미지 설명](https://example.com/image.png)
```

---

## 6. 코드 (Code)

### 6.1 인라인 코드

- 백틱(`) 1개 사용
    

```markdown
`print("Hello")`
```

### 6.2 코드 블록

- 백틱(```) 3개 사용
    
- 언어 지정 시 문법 강조 적용
    

````markdown
```python
print("Hello, World")
````

````

---

## 7. 인용문 (Blockquote)
- `>` 사용
- 중첩 가능

```markdown
> 인용문
>> 중첩 인용문
````

---

## 8. 수평선 (Horizontal Rule)

- `---`, `***`, `___` 사용 가능
    

```markdown
---
```

---

## 9. 표 (Tables)

- `|`로 열 구분
    
- `-`로 헤더 구분
    

```markdown
| 이름 | 나이 | 직업 |
|------|------|------|
| 홍길동 | 30 | 개발자 |
| 김철수 | 25 | 디자이너 |
```

---

## 10. 체크박스 (Task List)

- `- [ ]` : 미완료
    
- `- [x]` : 완료
    

```markdown
- [ ] 할 일
- [x] 완료된 일
```

---

## 11. 줄바꿈 (Line Break)

- 문장 끝 공백 2칸
    
- 또는 `<br>` 사용
    

```markdown
첫 번째 줄  
두 번째 줄

첫 번째 줄<br>
두 번째 줄
```

---

## 12. 이스케이프 (Escape)

- `\`로 특수문자 무력화
    

```markdown
\*기울임 아님\*
```

---

## 13. 기타

- 코드 내 백틱 표현: 백틱 3개 또는 4개 사용
    
- HTML 태그 일부 사용 가능 (`<br>`, `<img>` 등)
    
- GitHub Flavored Markdown(GFM)에서 확장 문법 지원
    

---

## 요약

- `#` : 제목
    
- `*`, `**` : 강조
    
- `-`, `1.` : 목록
    
- `[ ]( )` : 링크
    
- `` ` `` : 코드
    
- `>` : 인용
    
- `---` : 구분선
    
- `|` : 표
    
- `- [ ]` : 체크박스
