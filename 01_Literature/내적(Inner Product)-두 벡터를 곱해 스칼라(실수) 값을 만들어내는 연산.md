#Innerproduct #scala

기하학적으로 한 벡터를 다른 벡터로 투영(Projection)한 길이에 투영된 대상의 길이를 곱한 것과 같음
데이터의 유사도를 측정하거나 물리적 일(Work)를 계산할 때 사용

# 내적의 기하학적 의미: '그림자'와 '방향'
내적을 시각적으로 이해하면 한 벡터가 다른 벡터의 방향으로 얼마나 힘을 쓰고 있는가? 를 나타냄
- 투영(Projection) : 벡터 a를 백터 b 위로 수직으로 내린 그림자의 길이는 [^1]// a // cos𝞱 
	- 벡터 a가 벡터 b 위로 수직으로 투영했을 때 그림자 길이를 의미

[^1]: a벡터 노름(Norm) 코사인 세터,  절대값 에이 코사인 세터

# 유사도 측정
- 내적이 양수  : 두 벡터가 예각  0~90도를 이루며 비슷한 방향을 향함
- 내적이 0 : 두 벡터가 직교(90도) (서로 상관관계가 없음)
- 내적이 음수 : 두 벡터가 둔각(90~100도)를 이루며 반대 방향을 향함
