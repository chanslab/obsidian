#환각 #Hallucination #RAG #ReAct 

Ai모델이 답변을 생성할 때 단순히 학습된 데이터에만 의존하지 않고 신뢰할 수 있는 외부 소스(문서, 데이터베이스, 웹 검색 결과)를 근거(Grounding)로 삼아 답변하는 방식

# 왜? 중요한가
- 최신성
	- 정보의 유효기간 문제를 해결
- 신뢰성
	- 출처를 제시하고 사용자가 확인 가능
- 정확성
	- 모르면 모른다고 답변하도록 제어 할 수 있음
# 핵심기술 (RAG)
- 검색
	- 사용자의 질문을 찾아냄
- 전달
	- 질문과 찾은 내용을 AI모델에 전달
- 생성
	- 전달 받은 내용을 바탕으로(Grounded) 답변을 작성
