#outerproduct 

두 벡터 a, b를 곱해 두 벡터 모두에 **수직인 방향** 을 향하는 새로운 벡터를 생성

| 구분      | 내적(Inner Product) | 외적(Outer Product) |
| ------- | ----------------- | ----------------- |
| 기호      | a･b               | a×b               |
| 결과값     | 스칼라(숫자)           | 벡터(방향과 크기 존재)     |
| 기하학적 의미 | 투영, 유사도(닮은 정도)    | 평면의 넓이, 수직인 방향    |
| 각도 관계   | cos𝞱 사용(직교하면 0)  | sin𝞱 사용 (평행하면 0) |
