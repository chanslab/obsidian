# SMOTE(Synthetic Minority Over-sampling Technique)
대표적인 오버샘플링 기법
단순히 소수 데이터를 복제하는 것이 아니라, 기존 데이터를 바탕으로 새로운 가상 데이터를 생성

# 새로운 가상 데이터 생성 원리(step-by-step)
1. 샘플 선택
	1. 소수 클래스 데이터 중 하나를 선택
2. 이웃 찾기
	1. 선택한 데이터와 가장 가까운 k개의 이웃(k-Nearest Neighbors)을 찾음
3. 선 긋기
	1. 이웃 중 하나를 무작위로 선택하여, 처음 선택한 데이터와 이웃 사이에 선을 긋고
4. 데이터 생성
	1. 그 선위의 임의의 지점에 새로운 데이터를 생성

#### SMOTE는 반드시 훈련 데이터에만 적용해야되며, 검증이나 테스트 데이터셋에 사용하면 정보가 유출되어 성능이 과하게 높게 나오는 왜곡 현상이 발생

#### 수치형 데이터: 기본적으로 거리 기반 알고리즘을 사용하므로, 범주형 데이터(성별, 지역 등)가 포함된 경우 SMOTE-NC와 같은 변형 기법을 사용해야 함
